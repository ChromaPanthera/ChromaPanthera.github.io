Pseudo Sudoku

Some details about the game on the right

Let's see if I know how to code a minigame.   It should be on the right side.   If it's not there, I failed.   If I remember, it should be doable with basic javascript.   The first and only game I made using Javascript was asteroids.   This one is a randomized version of Sudoku.

Rules:
Fill up the chart with numbers 1-9
A number should be unique inside the 3x3 box, outlined by the purple dividers; vertically; and horizontally

How to navigate the controls

Click the "New board" button to generate a new Sudoku puzzle.
Click on a number at the bottom of the board.   It will turn orange if it's selected.
Click on an empty square.
If it's green, the rules are followed.
If it's red, the number exist somewhere.
If you need to erase, click on it again with the same number.
There is no set solution which means 2 things:
1. Solution will not be unique.
2. If the puzzle ends up being unsolveable, sorry about that.

Target Practice

What is this?

This is a shooting game.
Your goal is to click and break the targets that pop up on screen.
There's a limited time that the target stays.
There's also a randomized pattern the targets appear.
At the end, there will be statistics of how well or poorly you did.

Controls

1. Click the Start button to begin.
2. There will be a 1.5 second delay to get you ready.
3. Targets appear, you click on them.
4. Endure for a minute.
5. Profit!

Math Marathon

What is it?

This is a math crunching game.
Math questions will appear on the board.
You have to answer it accurately.
Answer them correctly as long as possible.
The game ends when you fail or when you answer 100 questions correctly.
No calculator, no writing.  You are challenged purely on mental computations.
This was a daily activity I was given as a child.  You are given some mercy, my teacher set a time limit of 5 minutes for the whole class.
Never underestimate basic operations.  It helps daily life really effectively.

Controls

Setup is trickier and has lots of nuances
1. Operations by default is addition.  That can be changed with a button click.
2. Set whether the first number is single or double digit.  Default is single.
    a. If operations are division, then it sets the digits of your answer.
3. Set whether the second number is single or double digit.  Default is single.
    a. If operations are subtraction, if you set first to single and second to double, you will get meaningless questions.
4. Click start when you are ready.
5. When questions appear, type your answer in the textbox.
6. Survive 100 questions.
7. If you get a mistake, it ends immediately.
8. It will show how many are right and how long you took.
